== Description ==
Using this plugin we can convert InnoDB storage engine type to MyISAM . We always recommend backing up your MySQL database before using this plugin.

== Installation ==

1. Upload the 'innodb-to-myisam' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Plugin can be used from "InnoDB to MyISAM" menu in left panel.
